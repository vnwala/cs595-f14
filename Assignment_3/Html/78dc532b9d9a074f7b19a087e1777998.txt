			</h1>
										<div class="comments-link">
					<a href="https://mew.su/2014/07/twitorss-instrument-kotory-j-pozvolit-iz-vash/#respond" title="Прокомментировать запись &laquo;TwiToRSS &#8212; инструмент, который позволит из вашей Twitter ленты сделать RSS канал&raquo;"><span class="leave-reply">Добавить комментарий</span></a>				</div><!-- .comments-link -->

				<div class="entry-content">
			<h2 style="text-align: center;">[ <a href="http://twitter.mew.su/rss.php">ЖМАК</a> ]</h2>
<p>Сделал сервис, который позволит сделать из ленты в Твиттере RSS канал. Пока-что функционал немного скудноват. Все что можно делать &#8212; это парсить твиты из ленты. Немного позднее добавлю возможность парсить твиты из списков (Lists), Любимых твитов (Favourites) и из поиска.</p>
<p>Для чего его можно использовать? Например для заполнения кучи профилей в Twidium Accounter ну или кросспостинга из RSS в блог (учел возможность автообновления ленты каждые n-секунд).</p>
<p>Если нужно &#8212; сделаю все тоже самое в формате JSON ну и вообще &#8212; следите за обновлениями. <img src="https://mew.su/wp-includes/images/smilies/icon_smile.gif" alt=":)" class="wp-smiley" /> </p>

				<div class="entry-content">
			<p>Все гениальное просто &#8212; для округления до десятков можно использовать функции, которые уже есть в PHP по-умолчанию (ceil и round), но для поставленной задачи они не совсем хорошо подходят.</p>
<p>Можно использовать следующий код:<br />
<code><br />
$p = 199.99; // 199 долларов 99 центов<br />
echo ceil($p/10) * 10; // 200 долларов ровно<br />
</code></p>
<p>Если сменить 10 на 100 во второй строке- скрипт будет округлять до сотых.</p>
