   Lynx is the text web browser.

   This is the top level page for the Lynx software distribution site
   hosted by the Internet Software Consortium.

   The [5]current development sources have the latest version of Lynx
   available (development towards 2.8.9).
   The [6]main help page for lynx-current is online; the current [7]User
   Guide is part of the online documentation.

   The most recent stable release is [8]lynx2-8-8.

   Other resources include:
     * [9]ftp and http mirrors
     * [10]Mailing list archives
     * [11]pgp/gpg signatures
