						<h2 class="post-title">
							<a href="http://manvsrow.com/2014/09/29/the-cure-for-your-songwriting-symptoms/" rel="bookmark">The Cure For Your Songwriting&nbsp;Symptoms</a>
						</h2>
						<p style="text-align:center;"><a href="http://manvsrow.files.wordpress.com/2014/07/man-vs-row.jpg"><img class="size-medium wp-image-768 aligncenter" style="display:block;margin-left:auto;margin-right:auto;outline-width:1px;outline-style:solid;outline-color:#777777;resize:none;" src="http://manvsrow.files.wordpress.com/2014/07/man-vs-row.jpg?w=300&#038;h=300" alt="Man vs Row" width="300" height="300" /></a></p>
<h3><span style="color:#ff0000;">Songwriters are a sick bunch of folks.</span></h3>
<div>And, no, I don’t mean we’re crazy for doing what we do. (Okay, maybe a little, but that’s not what I’m talking about today.)  What I mean by sick is that we have a lot of aches and pains that really bother us. As a pro songwriter and teacher, I’ve had a lot of writers come to me over the years looking for a way to heal their pain.</div>
<h3><span style="color:#0000ff;">Songwriter pains include:</span></h3>
<h3><span style="color:#0000ff;"> *I’m not getting cuts.</span><br />
<span style="color:#0000ff;"> *I can’t find good cowriters.</span><br />
<span style="color:#0000ff;"> *Publishers won’t set me up with cowrites.</span><br />
<span style="color:#0000ff;"> *My ASCAP/BMI rep won’t introduce me to publishers or cowriters.</span><br />
<span style="color:#0000ff;"> *Publishers won’t give me a meeting.</span></h3>
<p>Songwriters are often tempted to treat each of these conditions as a separate illness, but I think that’s usually a mistake.</p>
<h2><span style="color:#ff0000;">These are just symptoms. The disease is having songs that aren’t good enough.</span></h2>
<p>We can treat the symptoms all day long, but until we address the disease itself, your songwriting success will be limited. For example, we might be able to smooze our backsides off and finally talk our way into a cowrite with a pro writer… but so what? He or she will figure out pretty quickly that our songwriting isn’t up to snuff, and we probably won’t get a second cowrite.</p>
<p>But if we write strong songs, it’ll be a lot easier to get a pro to sit down to write. And when we prove our skill again- in the writing room- it’ll be easier to get a second cowrite.</p>
<p>It’s hard to think of a “songwriter symptom” that isn’t dramatically helped or fixed completely by writing better songs.<br />
<strong>Want your ASCAP or BMI rep to start introducing you around? <span style="color:#0000ff;">Write better songs.</span></strong><br />
<strong> Want better song evaluations? <span style="color:#0000ff;">Write better songs.</span></strong><br />
<strong> Want better cowrites? <span style="color:#0000ff;">Be a better cowriter yourself- by writing better songs.</span></strong><br />
<strong> Want cuts? <span style="color:#0000ff;">Write incredible songs, which lead to more relationships in the biz which lead to more opportunities to write more incredible songs, which put you in a much better position to get cuts.</span></strong></p>
<p>Treat the illness and the symptoms become a lot less severe or just go away on their own.</p>
<p>God Bless,</p>
<p>Brent</p>

<p>Welcome to Wordplay Thursday!</p>
<p>Here&#8217;s a writing prompt for you.  It&#8217;s a simple fill-in-the-blank. You can use one word or several. Feel free to get as crazy, genre-appropriate, or as imaginative as you want. The point is to get the creative juices flowing. And it&#8217;s a good thing to dig deeper, so don&#8217;t stop at the first idea that hits you. Try coming up with at least five things.</p>
<p>&#8220;My heart beat faster than _____.&#8221;</p>
<p>I&#8217;ll give you an example to get you started:</p>
<p>&#8220;My heart beat faster than <span style="text-decoration:underline;"><strong>hummingbird wings</strong></span>.&#8221;</p>
<p>I&#8217;d love to hear what you come up with, so please share in the comments. Oh, and please keep your posts below an R-rating. It&#8217;s a family show, after all!</p>

						<h2 class="post-title">
							<a href="http://manvsrow.com/2014/09/22/dont-fall-for-the-facebook-fantasy/" rel="bookmark">Don&#8217;t Fall For The Facebook&nbsp;Fantasy</a>
						</h2>
						<p style="text-align:center;"><a href="http://manvsrow.files.wordpress.com/2014/07/man-vs-row.jpg"><img class="size-medium wp-image-768 aligncenter" style="display:block;margin-left:auto;margin-right:auto;outline-width:1px;outline-style:solid;outline-color:#777777;resize:none;" src="http://manvsrow.files.wordpress.com/2014/07/man-vs-row.jpg?w=300&#038;h=300" alt="Man vs Row" width="300" height="300" /></a></p>
<div>
<h3><span style="color:#ff0000;">It’s easy to get frustrated by facebook.</span></h3>
<p>You scroll through your feed, and it seems like everyone in Nashville is:</p>
<h3><span style="color:#008000;"> “Writing a hit with ___ today!”</span></h3>
<h3><span style="color:#008000;"> “Listening to new mixes – they sound awesome!”</span></h3>
<h3><span style="color:#008000;"> “Just booked a show in ___!”</span></h3>
<h3><span style="color:#008000;"> “Just got a cut today- wish I could tell you who, but I will soon!”</span></h3>
<p>And it may all be true. I try to listen to my better angels and be happy for the success of others, even when I’m not getting as many cuts as I want.</p>
<p>But just remember- you only see what other folks WANT you to see. Certain people are going to overhype everything in an attempt to fake it till they make it. Others are overconfident or excited and really think every song they write is a hit. Others are so insecure that they feel they have to make things sound better than they are so they don’t get outted as the frauds they fear they are. Others are completely honest, but they only post the positive things.</p>
<h3><span style="color:#0000ff;">All this can lead to the false impression that everyone is on the fast lane of success while you can’t get out of the garage.</span></h3>
<p>So, how do we avoid falling into the facebook fantasy? First of all, don’t spend too much time on it. There’s probably something better you can do with your time, anyway. (Yes, I’ll post this to my facebook page, and I’m well aware of the irony.) Secondly, realize that the truth behind most of those posts is probably:</p>
<h3><span style="color:#008000;">“Writing a hit with ___ today… and my publisher probably won’t even like it so it’ll never even get demoed!”</span></h3>
<h3><span style="color:#008000;"> “Listening to new mixes- they sound awesome… and every label will pass on the song at least twice!”</span></h3>
<h3><span style="color:#008000;"> “Just booked a show in ___ … and after gas and paying the band, it’ll cost me $100!”</span></h3>
<h3><span style="color:#008000;"> “Just got a cut today- wish I could tell you who, but I will soon… as I google them because it’s some no-name indie artist that probably won’t even pay mechanicals on the 1,000 units they’ll print, anyway!”</span></h3>
<p>While that’s kind of a fun game, don’t be a hater. But just as importantly, don’t let facebook make you hate yourself.<br />
Now, if you’ll excuse me… I need to go write a hit and listen to some sweet new mixes.</p>
<p>God Bless,</p>
<p>Brent</p>

<p>Welcome to Wordplay Thursday!</p>
<p>Here&#8217;s a writing prompt for you.  It&#8217;s a simple fill-in-the-blank. You can use one word or several. Feel free to get as crazy, genre-appropriate, or as imaginative as you want. The point is to get the creative juices flowing. And it&#8217;s a good thing to dig deeper, so don&#8217;t stop at the first idea that hits you. Try coming up with at least five things.</p>
<p>&#8220;I felt more out of place than _____.&#8221;</p>
<p>I&#8217;ll give you an example to get you started:</p>
<p>&#8220;I felt more out of place than <span style="text-decoration:underline;"><strong>a number in the alphabet</strong></span>.&#8221;</p>
<p>I&#8217;d love to hear what you come up with, so please share in the comments. Oh, and please keep your posts below an R-rating. It&#8217;s a family show, after all!</p>

						<h2 class="post-title">
							<a href="http://manvsrow.com/2014/09/15/songwriters-take-your-listeners-to-the-movies/" rel="bookmark">Songwriters, Take Your Listeners To The&nbsp;Movies</a>
						</h2>
						<p style="text-align:center;"><a href="http://manvsrow.files.wordpress.com/2014/07/man-vs-row.jpg"><img class="alignnone size-medium wp-image-768" src="http://manvsrow.files.wordpress.com/2014/07/man-vs-row.jpg?w=300&#038;h=300" alt="Man vs Row" width="300" height="300" /></a></p>
<div>
<p><span style="color:#008000;"><strong>Brent is a hit songwriter with cuts by Alan Jackson, Randy Travis, Lady Antebellum, Joe Nichols, Gord Bamford, Ray Stevens, and more.  He&#8217;s written a top 5 hit in the US and a #1 in Canada&#8230; so far.</strong></span></p>
<p>Imagine yourself in a darkened movie theater. The movie starts to play, but there is just sound… no picture. You’d be upset, right? Well, then, why do we sometimes write songs that way?</p>
<p>I think the movie analogy is an appropriate one for songwriting.</p>
<h2><span style="color:#0000ff;">After all, don’t we basically write 3-minute movies?</span></h2>
<p>Our job is to entertain, to move, or to make the listener think. Just like a movie. But because songs are an audio format, we sometimes forget about the pictures. But they are terribly important!</p>
<p>Take, for instance, “The Thunder Rolls” written by Garth Brooks and Pat Alger. Yes, it’s an oldie, but it’s a classic. This lyric is a movie all by itself. Let’s look at the first verse:</p>
<p><em>3:30 in the morning / not a soul in sight / the city’s looking like a ghost town / on a moonless summer night / raindrops on the windshield / there’s a storm moving in / he’s heading back from somewhere / that he never should have been / and the thunder rolls</em></p>
<p>You can SEE that verse. The ghost town, the dark night, the raindrops. Not only that, but you can HEAR it. The thunder rolls. While this lesson will focus on visuals, don’t forget that you have FIVE senses, and you should use as many of them in a song as possible. Let’s look at the second verse:</p>
<p><em>Every light is burning / in a house across town / she’s pacing by the telephone / in her faded flannel gown / askin’ for a miracle / hopin’ she’s not right / praying it’s the weather / that’s kept him out all night / and the thunder rolls</em></p>
<p>Again, you can SEE and HEAR that verse. Lights burning, pacing by the phone, the faded flannel gown, the thunder rolls. And the third verse is just as visual as the first two.</p>
<p>It is no accident that some writers refer to sensory details as “furniture.” An empty room is not very inviting. It doesn’t hold your attention very long. However, a room with a great big couch and great art on the walls INVITES you in for a while. It gives you something to look at.</p>
<p>I got this feedback from an old publisher when I didn’t have strong visuals in a song. He said it left him, as he called it, “floating around in space with nothing to hang on to. You’re just telling me how you FEEL.”</p>
<h2>There’s a songwriting adage that says, <span style="color:#0000ff;">“Don’t TELL me, SHOW me.”</span> Visuals give you something to latch on to. A strong visual or other sensory image at the front end of a song really draws a listener in. It gives you a picture right off the bat.</h2>
<p>Take these following first lines from some recent hit songs:</p>
<p><em>Doublewide Quick Stop midnight T-top Jack in her Cherry Coke town</em> &#8211; “American Kids” sung by Kenny Chesney</p>
<p><em>Quarter in the payphone, clothes drying on the line</em> &#8211; “Automatic” sung by Miranda Lambert</p>
<p><em>Those high heels with that sun dress, turquoise heart hanging ‘round your neck</em> &#8211; “My Eyes” sung by Blake Shelton</p>
<p><em>Summer comin’ through a rolled down window, tearin’ down an almost two lane back road</em> &#8211; “We Are Tonight” sung by Billy Currington</p>
<p>And now a few hits that are a couple years back&#8230;</p>
<p><em>Sun shines, clouds rain, train whistles blow and guitars play</em> – “It Just Comes Natural” sung by George Strait</p>
<p><em>I’ve packed a cooler and a change of clothes</em> – “Want To” by Sugarland</p>
<p><em>Driving through town, just my boy and me. With a happy meal on his booster seat</em>- “Watching You” by Rodney Atkins</p>
<p><em>I can take the rain on the roof of this empty house</em>- “What Hurts The Most” Rascal Flatts</p>
<p><em>She’s a yellow pair of running shoes, a holey pair of jeans</em>- “She’s Everything” Brad Paisley</p>
<p>I could do this for days. Now, I know there are examples out there of purely emotional songs that do well. But if you look at the songs that are not written by the artist or by the producer or by an established hit songwriter, I think you’ll see a trend.</p>
<h2>So good luck with your songwriting. Use lots of visuals, and keep at it.</h2>
<p><strong>What about you?  Do you tend to write with or without a lot of imagery?  Are there lines from some other songs you think have great imagery that you&#8217;d like to share?  I&#8217;d love to hear from you!</strong></p>
<p>God Bless,</p>
<p>Brent</p>

<p>Welcome to Wordplay Thursday!</p>
<p>Here&#8217;s a writing prompt for you.  It&#8217;s a simple fill-in-the-blank. You can use one word or several. Feel free to get as crazy, genre-appropriate, or as imaginative as you want. The point is to get the creative juices flowing. And it&#8217;s a good thing to dig deeper, so don&#8217;t stop at the first idea that hits you. Try coming up with at least five things.</p>
<p>&#8220;August was hotter than_____.&#8221;</p>
<p>I&#8217;ll give you an example to get you started:</p>
<p>&#8220;August was hotter than <span style="text-decoration:underline;"><strong>biscuits in a cast iron skillet</strong></span>.&#8221;</p>
<p>I&#8217;d love to hear what you come up with, so please share in the comments. Oh, and please keep your posts below an R-rating. It&#8217;s a family show, after all!</p>

						<h2 class="post-title">
							<a href="http://manvsrow.com/2014/09/08/are-you-ready-to-know-music-row/" rel="bookmark">Are You Ready To Know Music&nbsp;Row?</a>
						</h2>
						<p style="text-align:center;"><a href="http://manvsrow.files.wordpress.com/2014/07/man-vs-row.jpg"><img class="size-medium wp-image-768 aligncenter" style="display:block;margin-left:auto;margin-right:auto;outline-width:1px;outline-style:solid;outline-color:#777777;resize:none;" src="http://manvsrow.files.wordpress.com/2014/07/man-vs-row.jpg?w=300&#038;h=300" alt="Man vs Row" width="300" height="300" /></a></p>
<div>
<p> If you have the dream of breaking into the music business as a songwriter or singer/songwriter, you’ve probably run into the following problems&#8230; or you will&#8230;</p>
<p><em>“I can’t get a PRO (ASCAP, SESAC, BMI) representative to help me. Publishers won’t listen to my songs. I can’t get a second meeting with my PRO or a publisher. Publishers will meet with me, but they won’t set me up with cowrites with their writers. What IS a PRO, anyway? And how can they help me?”</em></p>
<p>You can spend a lot of time guessing the answers or asking other songwriters who are in the same situation as you. The process of trial-and-error is really frustrating- you’re spending time trying to figure out how to get help from your PRO or a publisher when you’d rather be writing songs or doing something else that feels like you’re actually making progress.</p>
<h3><strong><span style="color:#ff6600;">PROs and publishers are an on-ramp to the music business highway, but we often find the on-ramp blocked by orange cones!</span></strong></h3>
<p>The day my PRO rep picked up the phone and called a publisher on my behalf was a big day in my career. He used his credibility and personal relationship with that publisher to open a door for me. It was a door I’d been knocking on but couldn’t open by myself. I ended up signing my first publishing deal with that same company, Major Bob Music.</p>
<h2><span style="color:#ff0000;">On Monday, September 22, Man vs. Row presents “Know The Row featuring Chad Green.”</span></h2>
<p>Chad spent 11 years as a Nashville Membership Representative for ASCAP (one of the three PROs &#8211; Performing Rights Organizations- in the US). He was responsible for signing numerous hit songwriters and publishers. He also oversaw the educational seminars for the Nashville office and created and implemented the successful ASCAP Writers Series workshops.</p>
<p><strong>Chad was also the ASCAP rep who picked up his phone for me that day.</strong></p>
<p>Chad has also spent years as a publisher. Currently, he is the Senior Creative Director for Daywind Music Publishing, where he oversees the day-to-day management of their staff of award-winning writers and producers focusing on multiple genres including Christian, Country, and Film/TV. Green is also responsible for the re-launch of Daywind&#8217;s Vital Records. Prior to his time with Daywind, he served as Creative Director for Word Music Publishing, a Warner/Curb company.</p>
<p>Chad is a real-deal publisher and PRO rep who has a heart to teach songwriters about the music business.</p>
<h3><strong><span style="color:#0000ff;">In this intimate, one-hour Google Hangout, Chad Green and hit songwriter Brent Baxter (“Monday Morning Church” by Alan Jackson) will discuss topics such as:</span></strong></h3>
<h3><span style="color:#0000ff;">*How to get PRO and publisher meetings.</span><br />
<span style="color:#0000ff;"> *How to get the most from your PRO.</span><br />
<span style="color:#0000ff;"> *What makes a songwriter stand out to a PRO or publisher.</span><br />
<span style="color:#0000ff;"> *The dos and don’ts of PRO and publisher meetings.</span><br />
<span style="color:#0000ff;"> *and more&#8230;</span></h3>
<p>But more than just being a fly on the wall, you’re invited to join in the conversation. Ask YOUR most burning questions to two industry professionals.</p>

						<h2 class="post-title">
							<a href="http://manvsrow.com/2014/09/08/single-of-the-year-gord-bamford-when-your-lips-are-so-close/" rel="bookmark">Single Of The Year: Gord Bamford, &#8220;When Your Lips Are So&nbsp;Close&#8221;</a>
						</h2>
						<p style="text-align:center;"><a href="http://manvsrow.files.wordpress.com/2013/09/gord-when-lips-single.jpg"><img class="alignnone size-full wp-image-227" src="http://manvsrow.files.wordpress.com/2013/09/gord-when-lips-single.jpg?w=538" alt="Gord When Lips Single"   /></a></p>
<div>
<p>Hey, ya&#8217;ll!</p>
<p>I just wanted to give you an update on my song, &#8220;When Your Lips Are So Close,&#8221; cut by Canadian artist, Gord Bamford.  Since it was released as the lead-off single from Gord&#8217;s &#8220;Country Junkie&#8221; album in September 2013, it&#8217;s had a great run.  It hit #1 in November of 2013, has been certified as a gold single, and was nominated for two Canadian Country Music (CCMA) Awards for Single Of The Year and Songwriters Of The Year.  While it didn&#8217;t win Songwriters Of The Year, it DID win Single Of The Year at last night&#8217;s CCMA Awards!  Congrats and thanks to Gord and Byron Hill, our cowriter and the coproducer on the project!</p>
<p>With the win last night, I thought I&#8217;d re-post this Cut Study for the song.  Enjoy!</p>
<p>God Bless,</p>
<p>Brent</p>
<h2><span style="color:#0000ff;">ORIGINAL POST:</span></h2>
<p>I’m blessed to have written Gord Bamford’s new single, “When Your Lips Are So Close.”  Gord is a hit artist on Sony Canada who is nominated for seven Canadian Country Music Awards.  If all goes according to plan, he’s going to debut our song on the 2013 CCMAs.  Today, I’d like to briefly discuss how I got the song cut and what you can learn from it.</p>
<h2><span style="color:#0000ff;">I networked my way to the artist.</span></h2>
<p>I started writing with hit songwriter, Byron Hill, back in 2004.  It wasn’t until later that Byron became Gord’s producer.  Because Byron and I had written several good songs together and he trusted my skill and work ethic, he felt comfortable bringing me into a cowrite with him and Gord.</p>
<h2><span style="color:#0000ff;">I asked for the cowrite.</span></h2>
<p>Byron and I are buddies, but he and Gord had a good thing going before bringing me in.  I already knew Byron pretty well, but I still used a patient approach to getting in the room.  It took months, but that’s okay.  You want to be persistent, but you don’t want to push too hard and make your contact uncomfortable.  You don’t want to lose the contact.</p>
<h2><span style="color:#0000ff;">I did my research.</span></h2>
<p>When Byron said he’d hook up a cowrite with him and Gord, I got copies of all of Gord’s records.  I listened and wrapped my head around Gord’s brand- what he likes to sing about and his lyrical “voice.”  I also talked to Byron about what works well for Gord.</p>
<h2><span style="color:#0000ff;">I did my pre-writing.</span></h2>
<p>I started an idea called “On My Best Days” and tailored it for Gord.  Gord and Byron liked the idea and the sketch I brought in, and we finished it.  It was an album cut on Gord’s album, “Is It Friday Yet?”</p>
<h2><span style="color:#0000ff;">I didn’t get lazy.</span></h2>
<p>Even though they cut the only song we’d written, I didn’t assume they’d call me up when Gord was writing for his next record.  Every once in a while, I mentioned to Byron how much I’d love to get back with Gord when he was in town to write.  After several months (and a couple of cancelled trips), we were back on the books.</p>
<h2><span style="color:#0000ff;">I did more research and pre-writing.</span></h2>
<p>I didn’t assume that Gord wanted or needed more of the same.  I asked Byron what they wanted for Gord’s next album.  Based on what Byron told me, I spent a few hours on my own looking through my ideas and adapting a few for Gord.  I ran them by Byron, and he liked two of them, “When Your Lips Are So Close” and “Nights Like You.”</p>
<h2><span style="color:#0000ff;">I focused on the artist’s needs.</span></h2>
<p>Byron and I originally thought “Nights Like You” would be a midtempo, but Gord liked it as a ballad- he thought it would really connect well with his audience.  Well, he’s the successful artist, and nobody knows his fans like he does, so I’m not going to argue with that.  (He cut that song, as well.)  We also worked to make sure “When Your Lips Are So Close” fit where Gord wanted to go and sat really well in his voice.  Thankfully, it worked out.</p>
<p>So, three cowrites with the artist and producer, and three cuts and one single- I’ll take it!  (I wish all my other artist cowrites worked out this well.)  So, here’s what I learned from this experience:</p>
<p>Use patience and persistence in your networking.  Do your research on the artist.  Pre-write.  Don’t get lazy.  Focus on the artist’s needs.  Of course, there are never any guarantees in the music business, but I believe this process gives you a better chance of success.  Good luck!</p>
</div>

<p>Thank you so much, Byron Hill and Gord Bamford, for writing with me and doing such a great job on our songs!  I&#8217;m honored to be a small part of your success!</p>

<p>Welcome to Wordplay Thursday!</p>
<p>Here&#8217;s a writing prompt for you.  It&#8217;s a simple fill-in-the-blank. You can use one word or several. Feel free to get as crazy, genre-appropriate, or as imaginative as you want. The point is to get the creative juices flowing. And it&#8217;s a good thing to dig deeper, so don&#8217;t stop at the first idea that hits you. Try coming up with at least five things.</p>
<p>&#8220;Dr. Phillips has the bedside manner of _____.&#8221;</p>
<p>I&#8217;ll give you an example to get you started:</p>
<p>&#8220;Dr. Phillips has the bedside manner of <span style="text-decoration:underline;"><strong>a prison guard</strong></span>.&#8221;</p>
<p>I&#8217;d love to hear what you come up with, so please share in the comments. Oh, and please keep your posts below an R-rating. It&#8217;s a family show, after all!</p>

							<a href="http://manvsrow.com/2014/09/03/are-you-ready-to-join-man-vs-row-in-the-top-10/" rel="bookmark">Are You Ready To Join Man vs. Row In The Top&nbsp;10?</a>
						</h2>
						<p style="text-align:center;"><a href="http://manvsrow.files.wordpress.com/2014/07/man-vs-row.jpg"><img class="size-medium wp-image-768 aligncenter" style="display:block;margin-left:auto;margin-right:auto;outline-width:1px;outline-style:solid;outline-color:#777777;resize:none;" src="http://manvsrow.files.wordpress.com/2014/07/man-vs-row.jpg?w=300&#038;h=300" alt="Man vs Row" width="300" height="300" /></a></p>
<div>
<h2><strong><span style="color:#ff0000;">The listener&#8217;s reaction to your song is only as real as the characters in your song.</span></strong></h2>
<p>Do you have trouble getting your songs to connect with your audience on an emotional level? Do the characters in your songs sometimes feel flat, cliche, or vague? Do you ever wonder how some characters in hit songs seem so real that they leap out of the radio and into your heart while your characters often don’t?</p>
<p>If your goal is success in the music business (as a songwriter writing for cuts, a singer/songwriter working toward a career on radio, or an indie artist wanting better shows and better sales at the merch table), then making your characters <strong>REAL</strong> is a big deal. If that never happens, odds are your career never happens, either. At least not as big as it should.</p>
<p><span style="color:#0000ff;"><strong>Usually when a listener doesn’t connect with the character in a song, it’s because the songwriter hasn’t PROVEN the character to the listener. The listener doesn’t care about your character because the character doesn’t feel real.</strong></span></p>
<h2><strong><span style="color:#ff9900;">The solution to this problem is usually imagery.</span></strong></h2>
<p>When used well, imagery makes your character believable. <strong>Pictures are proof.</strong> Imagery gets your song past the listener’s head and into the listener’s heart.</p>
<p>TELL me about an cute little abandoned shelter puppy, and I’ll be sad for a minute. SHOW me that cute little abandoned shelter puppy, and I’ll adopt it.</p>
<p>I’m a lyricist. Words are my thing. In a town full of incredible musicians and great singers, I’ve been able to land publishing deals (Major Bob, Peer Music, etc.), cuts (Joe Nichols, Lady Antebellum, Randy Travis, etc.), and hits (Alan Jackson, Gord Bamford) as a lyricist. And a big part of being a successful lyricist is knowing how to use imagery to create believable characters.</p>
<h3><strong><span style="color:#ff0000;">On Wednesday, September 17, from 8pm to 9pm Central, I’ll be hosting an exclusive online workshop to teach you how to use imagery to bring your characters to life.</span></strong></h3>
<p>There are only a few “Top Ten” tickets still available for this event, so don&#8217;t wait.  Click on the image below to find out more!</p>

			function JetpackLikesWidgetQueueHandler() {
				var wrapperID;
				if ( ! jetpackLikesMasterReady ) {
					setTimeout( JetpackLikesWidgetQueueHandler, 500 );
					return;
				}

				if ( jetpackLikesWidgetQueue.length > 0 ) {
					// We may have a widget that needs creating now
					var found = false;
					while( jetpackLikesWidgetQueue.length > 0 ) {
						// Grab the first member of the queue that isn't already loading.
						wrapperID = jetpackLikesWidgetQueue.splice( 0, 1 )[0];
						if ( jQuery( '#' + wrapperID ).hasClass( 'jetpack-likes-widget-unloaded' ) ) {
							found = true;
							break;
						}
					}
					if ( ! found ) {
						setTimeout( JetpackLikesWidgetQueueHandler, 500 );
						return;
					}
				} else if ( jQuery( 'div.jetpack-likes-widget-unloaded' ).length > 0 ) {
					// Grab any unloaded widgets for a batch request
					JetpackLikesBatchHandler();

					// Get the next unloaded widget
					wrapperID = jQuery( 'div.jetpack-likes-widget-unloaded' ).first()[0].id;
					if ( ! wrapperID ) {
						// Everything is currently loaded
						setTimeout( JetpackLikesWidgetQueueHandler, 500 );
						return;
					}
				}

				if ( 'undefined' === typeof wrapperID ) {
					setTimeout( JetpackLikesWidgetQueueHandler, 500 );
					return;
				}

				var $wrapper = jQuery( '#' + wrapperID );
				$wrapper.find( 'iframe' ).remove();

				if ( $wrapper.hasClass( 'slim-likes-widget' ) ) {
					$wrapper.find( '.post-likes-widget-placeholder' ).after( "<iframe class='post-likes-widget jetpack-likes-widget' name='" + $wrapper.data( 'name' ) + "' height='22px' width='68px' frameBorder='0' scrolling='no' src='" + $wrapper.data( 'src' ) + "'></iframe>" );
				} else {
					$wrapper.find( '.post-likes-widget-placeholder' ).after( "<iframe class='post-likes-widget jetpack-likes-widget' name='" + $wrapper.data( 'name' ) + "' height='55px' width='100%' frameBorder='0' src='" + $wrapper.data( 'src' ) + "'></iframe>" );
				}

				$wrapper.removeClass( 'jetpack-likes-widget-unloaded' ).addClass( 'jetpack-likes-widget-loading' );

				$wrapper.find( 'iframe' ).load( function( e ) {
					var $iframe = jQuery( e.target );
					$wrapper.removeClass( 'jetpack-likes-widget-loading' ).addClass( 'jetpack-likes-widget-loaded' );

					JetpackLikespostMessage( { event: 'loadLikeWidget', name: $iframe.attr( 'name' ), width: $iframe.width() }, window.frames[ 'likes-master' ] );

					if ( $wrapper.hasClass( 'slim-likes-widget' ) ) {
						$wrapper.find( 'iframe' ).Jetpack( 'resizeable' );
					}
				});
				setTimeout( JetpackLikesWidgetQueueHandler, 250 );
			}
